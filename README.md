# NYC Airbnb Data Analysis (Python)

## Research Question
How does the number of reviews relate to listing price, availability, and other features?

## Project Overview
This project performs an end-to-end data analysis on the NYC Airbnb Open Data dataset.

The workflow includes:
- Data cleaning
- Outlier treatment using IQR
- Exploratory Data Analysis (EDA)
- Correlation analysis
- Linear Regression modeling
- Random Forest modeling
- Business insights and recommendations

## Key Findings
- Price has very weak correlation with number_of_reviews
- Availability shows slight positive correlation
- Location and room type have stronger influence on review count
- Reviews are influenced by multiple hidden qualitative factors

## Tools Used
- Python
- pandas
- numpy
- seaborn
- matplotlib
- scikit-learn
